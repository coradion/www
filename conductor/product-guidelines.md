# Product Guidelines: Eudemonic Task Management SaaS

## Prose and Communication
- **Minimalist & Direct:** Copy must be surgical. Eliminate fluff. Focus on essential instructions and status updates. The interface should speak through action rather than explanation.

## Visual Aesthetic
- **Modern & Dynamic:** A sleek, high-performance visual style that feels responsive and fluid. Use high-quality typography and a professional color palette that emphasizes clarity and speed.

## User Experience (UX) Principles
- **Aggressive Elimination:** Every field, button, and visual element must justify its existence. If it doesn't directly contribute to the task at hand, remove it.
- **Progressive Disclosure:** Shield the user from cognitive overload by hiding advanced options and complex data until explicitly requested.

## Inclusivity and Accessibility
- **Neuro-Inclusive Design:** The system is explicitly engineered for users with ADHD and cognitive processing challenges. Avoid cluttered layouts, horizontal scanning, and unpredictable interactions. Focus on maintaining a single thread of attention.
